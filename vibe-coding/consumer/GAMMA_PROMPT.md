# Gamma 프레젠테이션 생성 프롬프트

다음 프롬프트를 Gamma (gamma.app)에 입력하여 이 프로젝트의 시연 프레젠테이션을 생성하세요:

---

## 프롬프트

**소비자물가지수 통계 분석 및 보도자료 자동 생성 시스템** 프로젝트 시연 프레젠테이션을 만들어줘.

### 슬라이드 구성:

**1. 타이틀 슬라이드**
- 제목: "소비자물가지수 통계 분석 시스템"
- 부제목: "KOSIS API 기반 실시간 데이터 분석 및 보도자료 자동 생성"
- 시각적 요소: 통계 차트나 데이터 아이콘

**2. 프로젝트 개요**
- 문제 정의: 통계청 소비자물가지수 데이터의 수동 처리 및 분석 작업의 비효율성
- 해결 방안: 실시간 API 연동을 통한 자동화된 통계 분석 시스템
- 목표 사용자: 통계 분석가, 경제 정책 연구원, 언론 보도 담당자, 정부 기관 공무원

**3. 핵심 기능 소개**
- **KOSIS API 실시간 연동**: 통계청 공식 API를 통한 자동 데이터 수집
- **10가지 주요 통계량 자동 계산**: 
  * 최근 3개월 평균 증가율
  * 최고/최저 상승률 달 분석
  * 물가 상승 추세 (선형 회귀 분석)
  * 변동성 지수 및 안정성 점수
  * 지출목적별 변동성 분석
  * 최근 6개월 변화 추세
  * 계절성 패턴 분석
- **Excel 자료 다운로드**: 원본 데이터를 Excel 형식으로 다운로드
- **보도자료 자동 생성**: 전문적인 HTML 형식 보도자료 자동 생성

**4. 기술 스택**
- **Backend**: Flask 3.0 (Python)
- **Data Processing**: pandas, numpy
- **API Integration**: KOSIS OpenAPI (requests)
- **Frontend**: HTML5, CSS3, JavaScript (ES6+)
- **File Generation**: openpyxl (Excel), HTML 템플릿

**5. 시스템 아키텍처**
- 사용자 인터페이스 (웹 브라우저)
- Flask 백엔드 서버
- KOSIS API 연동
- 데이터 분석 엔진 (pandas, numpy)
- 보도자료 생성 엔진

**6. 주요 통계량 상세 설명**
각 통계량을 카드 형식으로 시각화:
- 최근 3개월 평균 증가율: 전년동월비 기준 물가 상승률 추이
- 최고/최저 상승률 달: 물가 변동의 극값 분석
- 물가 상승 추세: 선형 회귀를 통한 장기 추세 파악
- 변동성 지수: 표준편차 기반 물가 안정성 측정
- 지출목적별 분석: 카테고리별 변동성 비교
- 물가 안정성 점수: 0-100점 척도의 안정성 평가
- 최근 6개월 변화: 단기 추세 분석
- 계절성 패턴: 월별 평균 물가지수 분석

**7. 사용자 인터페이스**
- 직관적인 기간 선택 (최근 N개월 / 날짜 범위)
- 실시간 통계 분석 결과 표시
- 카드 형식의 통계량 시각화
- 반응형 디자인 (데스크톱, 태블릿, 모바일 지원)
- 보라색 그라데이션 디자인

**8. 보도자료 생성 기능**
- 공식 문서 형식의 HTML 보도자료
- 일러두기, 주요 통계량 분석, 지출목적별 분석, 종합 분석 포함
- 인쇄 최적화된 레이아웃
- 자동 날짜 및 시간 정보 삽입

**9. 데이터 흐름**
1. 사용자가 기간 선택 (최근 N개월 또는 날짜 범위)
2. Flask 서버가 KOSIS API 호출
3. 데이터를 pandas DataFrame으로 변환
4. 10가지 통계량 자동 계산
5. 결과를 JSON으로 반환하여 프론트엔드에 표시
6. 보도자료 생성 요청 시 HTML 형식으로 변환

**10. 실제 사용 시나리오**
- 시나리오 1: 통계 분석가가 최근 36개월 데이터를 분석하여 주요 인사이트 도출
- 시나리오 2: 언론 담당자가 월간 보도자료를 자동 생성하여 배포
- 시나리오 3: 정책 연구원이 특정 기간(예: 2020-2023)의 물가 추세를 분석

**11. 기술적 특징**
- API 실패 시 로컬 Excel 파일로 자동 폴백
- 다양한 API 응답 형식 처리 (리스트/딕셔너리)
- 에러 핸들링 및 사용자 친화적 오류 메시지
- 환경 변수를 통한 API 키 보안 관리

**12. 성능 및 호환성**
- API 응답 시간: 3초 이내
- 통계 계산 시간: 5초 이내
- 최신 브라우저 지원 (Chrome, Firefox, Safari, Edge)
- Python 3.8+ 지원

**13. 향후 개선 계획**
- 차트/그래프 시각화 추가
- 여러 기간 비교 분석 기능
- PDF 형식 보도자료 다운로드
- 이메일 자동 발송 기능
- 데이터베이스 연동 및 캐싱

**14. 데모 시연 포인트**
- 실시간 API 데이터 수집 과정
- 통계량 계산 결과 시각화
- 보도자료 자동 생성 결과물
- Excel 파일 다운로드 기능

**15. 마무리 슬라이드**
- 프로젝트 요약
- 핵심 가치 제안: "수동 작업을 자동화하여 시간을 절약하고 일관된 분석 결과 제공"
- 연락처 또는 GitHub 링크 (선택사항)
